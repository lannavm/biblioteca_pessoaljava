# biblioteca_pessoaljava

Projeto pessoal para treinar minhas habilidades em Java. 

O projeto propõe a criação de uma aplicação para o controle da minha biblioteca pessoal. 

Objetivos principais:
  * Listar todos os livros, seus autores, edição, gênero e ano em que foram publicados.
  * Criar um menu com as seguintes opções:
    
        * Adicionar um livro
        * Remover um livro
        * Avaliar livro
        * Adicionar frase favoritas de cada livro
        * Editar informações
